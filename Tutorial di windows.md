# Cara Install dan penggunaan Terminalizer di Windows

## Install terminalizer

### 1. Pasang Node.js
- Download Node.js versi 22.21.1 LTS dari link berikut
  
  https://nodejs.org/dist/v22.21.1/node-v22.21.1-x64.msi
  
- Jalankan file installer.
- Ikuti proses instalasi sampai selesai.

### 2. Buka Command Prompt
- Tekan Windows + R.
- Ketik `cmd`
- Tekan Enter.

### 3. Install Terminalizer Secara Global
- Jalankan perintah berikut:
```
npm install -g terminalizer
```
### 4. Cek versi
```
terminalizer --version
```

## Penggunaan terminalizer

### 1. Cek versi
Masukan prompt

```
terminalizer --version
```

Kalo muncul hasilnya, lanjut step 2

### 2. Memulai Record Terminalizer

contoh:

```
terminalizer record rehan
```
**Nama rehan ganti sesuai kebutuhan**

**nanti akan mucul tab cmd baru nah pas munculnya tab baru itu sedang merekam prompt kalian**


### 3. Mengakhiri Record Terminalizer, render dan play hasilnya

ketik 

```
exit
```

```
terminalizer render rehan
```
